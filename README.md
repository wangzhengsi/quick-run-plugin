<h1>一款idea插件，在不需要代码入口的情况下随意调用java方法</h1>
<h2>使用方式：</h2>
<p>1. 运行项目。</p>
<p>2. 右键点击您要调用的方法。</p>
<p>3. 从菜单中选择<strong>“Run Method”</strong>。</p>
